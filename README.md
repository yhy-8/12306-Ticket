# 12306 抢票助手 Web 版

## 快速开始

### 依赖安装

```bash
pip install -r requirements.txt
```
### 启动

```bash
python main.py
```

### 使用
打开浏览器访问 `http://127.0.0.1:8080`


## 配置说明
- **多人抢票**：一个人扫码，从乘车人中选择多人抢票，每个人的**乘车信息及行程**由本地已保存的信息决定
- **信息保存**：点击右上角**小人图标**进入用户配置管理，信息会保存至 `data/user_data.json`
- **站点选择**：输入城市名称会自动过滤出对应的车站
- **车次列表**：多个车次请用英文逗号分隔，如 `G1,G102,D311`

### config.json文件具体说明
```
max_try_times = 3   #整体流程抢票尝试次数
logswitch = False   #是否开始日志记录，开启后会降低抢票速度，建议正常使用不开启
ischoose_position = True #是否启用座位/床位选择
advanced = 0.0     #提前发送请求时间，单位(s)。适当地提前可以规避网络延迟，过于提前会导致重复请求失败，达到五次会断开连接！
grabfunction_max_try_times = 6 #抢票各部分函数最多重复次数
```
